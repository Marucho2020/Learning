	## Thành phần chính 

	Remote Interface                 : Định nghĩa các method mà Client có thể gọi từ xa 
	Remote Object                    : Class triển khai Remote Interface  
	Stub (Client Proxy )             : Đại diện cho Remote Object trên Client giúp gửi Request 
	
	Skeleton(Server Proxy)           : Nhận request, gọi Remote Object thật (Từ Java 1.1 trở về trước )
	Reflection API(Server Proxy)     : Nhận request, gọi Remote Object thật  (Sau Java 1.2 )
	
	RMI Registry                     : Dịch vụ giúp đăng ký và tìm kiếm Remote Object 
	Marshalling/ Unmarshalling       : Chuyển đổi Object thành dữ liệu truyền qua mạng
