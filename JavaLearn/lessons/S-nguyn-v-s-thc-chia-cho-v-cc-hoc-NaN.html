Số nguyên và số thực chia cho vô cực hoặc NaN

	
		## Chia một số nguyên hoặc thực cho Infinity 
			10 / Infinity    =  0 
			10.0 / Infinity   = 0 
			-10 / Infinity   = 0.0 
			
		## Chia Bất kỳ số nào cho NaN kết quả luôn là NaN 
			10.0 / NaN = NaN 
			
		
	Tóm tắt:
		Kiểu Dữ Liệu	Phép Toán	Kết Quả
		Integer	x / 0	ArithmeticException
		Floating-point	x / 0.0 (x ≠ 0)	Infinity hoặc -Infinity
		Floating-point	0.0 / 0.0	NaN
		Integer	x % 0	ArithmeticException
		Floating-point	x % 0.0	NaN
