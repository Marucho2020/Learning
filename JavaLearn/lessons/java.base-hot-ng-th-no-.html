java.base hoạt động thế nào ?

	
	🔹 a) không cần requires java.base 
		- Khi tạo module, bạn không cần requires java.base vì nó được import mặc định  
		
				module mymodule {
					requires java.sql;  // Chỉ cần khai báo các module khác ngoài java.base
				}

	
	🔹 b) java.base không phụ thuộc vào module nào khác 
		- Tất cả các module khác trong JDK đều phụ thuộc vào java.base 
		- Nhưng java.base không phụ thuộc vào bất kỳ nhóm module nào khác 
	
	🔹 Ví dụ Dependency Tree của java.sql
	
			$ jdeps --module java.sql
			java.sql -> java.base
			java.sql -> java.logging

			➡ java.sql cần java.base để hoạt động.
