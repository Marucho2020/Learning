4. jshell (Java Shell )

	## Công dụng : 
		Là một REPL(Read-Eval-Print Loop) cho Java, cho phép bạn chạy mã Java trực tiếp mà không cần viết trong file .java 
		
	## Ứng dụng : 
		- Thử nghiệm nhanh các đoạn code java 
		- Học Java hoặc kiểm tra ý tưởng code mà không cần tạo file 
			
	## Cú pháp 
		jshell
		
		# Ví dụ , giống như chạy 1 chương trình python thử nghiệm k cần tạo các class main phức tạp  
		jshell
		jshell> int x = 10;
		jshell> System.out.println(x + 5);
		15


Tóm tắt dễ nhớ:
Lệnh	Chức năng chính
jdeps	Phân tích dependencies của ứng dụng.
jlink	Tạo runtime Java tùy chỉnh, chỉ chứa các module cần thiết.
javap	Xem chi tiết bytecode bên trong file .class.
jshell	Chạy thử mã Java trực tiếp (REPL).
