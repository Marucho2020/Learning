Các lớp DateTime quan trọng trong java.time

		
	
	Class : 

	LocalDate				// Lưu ngày (không có giờ, phút, giây)					// 2025-02-05
						
	LocalTime				// Lưu thời gian (không có ngày) 						// 14:30:15
							
	LocalDateTime			// Lưu ngày giờ và múi giờ 								// 2025-02-05T14:30:15
				
	ZonedDateTime			// Lưu ngày, giờ kèm theo múi giờ 						// 2025-02-05T14:30:15+07:00[Asia/Ho_Chi_Minh]
				
	Instant 				// Thời gian tính theo UNIX timestamp (UTC)				// 2025-02-05T07:30:15Z 
		
	Duration 				// Khoảng cách giữa hai thời điểm (giây, nano giây)		// PT3H30M (3 giờ 30 phút) 
		
	Period					// Khoảng cách giữa hai ngày (năm, tháng, ngày)			// P2Y5M3D (2 năm 5 tháng 3 ngày)
