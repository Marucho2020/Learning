Số thực chia cho 0

		## Khi một số thực float hoặc double chia cho 0 , Java sẽ KHÔNG ném ngoại lệ mà kết quả tuân theo quy tắc chuẩn của IEEE 754 
		 
		## Chia cho 0.0 
			10.0 / 0.0   -> Infinity (Dương vô cực)
			-10.0 / 0.0   -> -Infinity (Âm vô cực )
			
			
			
		## Chia 0.0 cho 0.0 
			0.0 / 0.0 -> NaN (Not a Number)
			10.0 % 0.0  -> NaN
