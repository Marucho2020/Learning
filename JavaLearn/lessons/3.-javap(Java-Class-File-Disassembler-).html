3. javap(Java Class File Disassembler )

	## Công dụng  : 
		Dùng để xem chi tiết bên trong file .class(bytecode). Nó giống như một công cụ giải mã file .class 
		
	## Ứng dụng : 
		Tìm hiểu cấu trúc bytecode của một class 
		Hỗ trợ debug lỗi liên quan đến complier 
		
	## Cú pháp 
		javap  
		
			Ví dụ : javap MyClass 
			
			Kết quả : Hiển thị cấu trúc class của MyClass, bao gồm các method, field, constructor 
			
	## Mở rộng 
		thêm option -c (hoặc gọi là thêm flag -c) để xem bytecode 
			javap -c MyClass
