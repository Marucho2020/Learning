1. jdeps(Java Dependency Analyzer) 依頼関係

	## Công dụng :
		Lệnh này dùng để phân tích các dependencies (phụ thuộc) trong một ứng dụng Java. Nó kiểm tra mã nguồn hoặc file .class/.jar phụ thuộc vào các thư viện nào 
		
	## Ứng dụng : 
		- Kiểm tra ứng dụng có phụ thuộc vào các API nội bộ hoặc deprecated không 
		- Giúp tối ưu hóa ứng dụng bằng cách loại bỏ thư viện không cần thiết 
		
	## Cú pháp 
		jdeps 
			
			jdeps myapp.jar
			Kết quả : Liệt kê tất cả các module và packages mà myapp.jar đang sử dụng
