2. jlink(Java Linker )

	
	## Công dụng 
		Tạo một runtime image tùy chỉnh, bao gồm những module cần thiết cho ứng dụng của bạn. 
	
	## Ứng dụng : 
		Tối ưu hóa dung lượng của ứng dụng 
		Đóng gói ứng dụng kèm runtime để chạy độc lập, không phụ thuộc vào cài đặt JDK/JRE 
		
	## Cú pháp  
		jlink --module-path  --add-modules <> --output 
		
			### Ví dụ 
				jlink --module-path $JAVA_HOME/jmods --add-modules java.base,java.sql --output mycustomjre
				
				Kết quả : tạo một runtime Java nhẹ với chỉ các module java.base và java.sql
