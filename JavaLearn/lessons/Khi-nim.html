Khái niệm

	Trong Java 9, hệ thống module được giới thiệu (JPMS - Java Platform Module System), và java.base là module lõi của JDK 
		- java.base chứa các API cơ bản nhất mà mọi ứng dụng Java cần 
		- Tự động được import vào tất cả các module, không thể bị loại bỏ 
		
		➡ Không có java.base thì JVM không thể hoạt động
