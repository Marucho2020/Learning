# 	Khái niệm 			
				
	Elasticity (tính đàn hồi) là khả năng hệ thống:			
		tự động tăng hoặc giảm tài nguyên theo nhu cầu thực tế		
				
				
	## Điểm quan trọng: 			
				
		- tự động
		- theo thời gian thực
		- trả tiền theo mức sử dụng



# Ví dụ thực tế 

	Một trang bán hàng.Traffic bình thường:
		1000 user

	Nhưng vào flash sale:
		100000 user


	AWS sẽ:
		10 servers → 100 servers

	Sau khi hết sale:
		100 servers → 10 servers

	Tự động.




# Ví dụ AWS

	Sử dụng AWS Auto Scaling 
		Khi CPU > 70% 
			→ thêm server
		
		Khi CPU < 30% 
			→ giảm server




-------------日本語------------


# **Elasticity（エラスティシティ）**とは、
	需要に応じて
		リソースを自動的に増減する能力



#例

	通常トラフィック
		1000 users

	セール時
		100000 users


	AWSは自動的に：
		10台 → 100台

	そして負荷が下がると
		100台 → 10台

		に戻ります。



# AWSサービス例

	主に使うのは
		AWS Auto Scaling

	条件例：
	
		CPU > 70% → サーバー追加
		CPU < 30% → サーバー削減
