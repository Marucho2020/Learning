# 		1. Bedrock Agent là gì? (Bản chất)
		
	Amazon Bedrock Agent là một AI Agent do AWS quản lý, có khả năng tự lập kế hoạch (planning), gọi các API (Action Group), truy xuất dữ liệu (Knowledge Base), ghi nhớ trạng thái hội thoại và tự quyết định các bước cần thực hiện để hoàn thành mục tiêu của người dùng.
		
		
	## Nếu chỉ có Claude (Foundation Model), quá trình sẽ là:	
		User
		   ↓
		Prompt
		   ↓
		Claude
		   ↓
		Trả lời		
		
	Claude chỉ suy nghĩ từ Prompt.	
		
	Nhưng trong thực tế doanh nghiệp:	
		
				
		Nhưng trong thực tế doanh nghiệp:

		Ví dụ:
			"Đặt vé máy bay từ Tokyo đến Osaka ngày mai."

		AI không thể chỉ "trả lời".

		Nó cần:
			tìm chuyến bay
			hỏi còn ghế không
			kiểm tra tài khoản
			thanh toán
			gửi email
			Claude không thể tự làm.
			Đó là lý do Bedrock Agent xuất hiện.		
			
			
Kiến trúc trở thành:		
		
		
		User
		   ↓
		Bedrock Agent
			  │
			  ├── Claude
			  │
			  ├── Knowledge Base
			  │
			  ├── Lambda
			  │
			  ├── API
			  │
			  ├── Database
			  │
			  └── nhiều service AWS khác		
				
		Agent giống như:
			Project Manager

		Claude giống:
			Nhân viên suy nghĩ

# 	8. Trade-off		
			
| Ưu điểm                    | Nhược điểm                                              |

# 12. Góc nhìn Master/Senior 

	Nhiều người mới học nghĩ:

	Bedrock Agent = Claude mạnh hơn.

	Đây là một hiểu lầm.

	Thực tế:

	Claude/Nova/Llama là "bộ não" (reasoning engine): nhận prompt và sinh nội dung.
	Bedrock Agent là "người điều phối" (orchestrator): quyết định khi nào dùng bộ não, khi nào truy xuất Knowledge Base, khi nào gọi API qua Action Group, và khi nào kết thúc workflow.

	Một cách hình dung:

	Foundation Model = Kỹ sư giỏi.
	Knowledge Base = Thư viện tài liệu.
	Action Group = Bộ công cụ để thao tác với hệ thống.
	Bedrock Agent = Quản lý dự án phân công công việc cho kỹ sư và sử dụng đúng công cụ để hoàn thành mục tiêu.

	Đây cũng là xu hướng hiện nay của AI doanh nghiệp: thay vì chỉ dùng LLM để "chat", các hệ thống đang chuyển sang Agentic AI, nơi AI có thể tự lập kế hoạch và thực hiện chuỗi hành động phức tạp. Bedrock Agent chính là dịch vụ AWS hiện thực hóa mô hình này.
