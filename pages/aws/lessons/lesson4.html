# Cloud là gì?

	Cloud (Điện toán đám mây) là mô hình cung cấp tài nguyên IT qua internet.
	Thay vì mua máy chủ và đặt trong công ty, bạn thuê tài nguyên từ nhà cung cấp cloud.

	## Các tài nguyên có thể bao gồm:

		- Server (máy chủ)
		- Storage (lưu trữ dữ liệu)
		- Database
		- Network
		- AI / Machine Learning
		- Monitoring / Security


	## Nhà cung cấp phổ biến:

		- Amazon Web Services (AWS)
		- Microsoft Azure
		- Google Cloud Platform



	## Ví dụ dễ hiểu:

		Trước đây nếu muốn chạy website bạn phải:
			-mua server
			-đặt trong phòng server
			-tự quản lý điện
			-tự sửa khi hỏng
			-Cloud thì khác:
			
			
	## Bạn chỉ cần tạo server trên internet trong vài phút.
			- Ví dụ trên AWS:
			- tạo server bằng Amazon EC2
			- lưu file bằng Amazon S3
			- Bạn trả tiền theo mức sử dụng (pay-as-you-go).


# 2️ On-premise là gì?

	On-premise nghĩa là toàn bộ hệ thống IT được đặt tại công ty của bạn.

	## Công ty phải tự:

		- mua server
		- lắp đặt datacenter
		- thuê kỹ sư vận hành
		- bảo trì phần cứng

		Nói thẳng:
		Bạn sở hữu mọi thứ và chịu mọi rủi ro.




# 3️⃣  So sánh Cloud vs On-premise


| Tiêu chí          | Cloud              | On-premise           |
| ----------------- | ------------------ | -------------------- |
| Chi phí ban đầu   | Thấp               | Rất cao              |
| Mở rộng hệ thống  | Rất dễ             | Khó                  |
| Triển khai server | Vài phút           | Vài tuần             |
| Bảo trì phần cứng | Nhà cung cấp cloud | Công ty tự làm       |
| Khả năng scale    | Tự động            | Phải mua thêm server |



4️⃣ Vì sao công ty chuyển sang Cloud?


	- 1️ Cost saving – tiết kiệm chi phí
	- 2️ Scalability – mở rộng dễ dàng
	- 3️ Speed – triển khai nhanh
	- 4️ Global infrastructure – server toàn cầu


5️ Ví dụ thực tế

	Ví dụ AWS có dịch vụ:
	
		AWS Auto Scaling

# Cloudとは何か

	**クラウド（Cloud Computing）**とは、
	インターネットを通してITリソースを提供するサービスのことです。

	会社が自分でサーバーを購入する代わりに、
	クラウドプロバイダーから必要な分だけ借りることができます。


	## 提供される主なサービス：

		- サーバー
		- ストレージ
		- データベース
		- ネットワーク
		- AI / Machine Learning

	## 代表的なクラウドプロバイダー： 
	
		- Amazon Web Services
		- Microsoft Azure
		- Google Cloud Platform	
	
	
	## AWSでは 
	
		仮想サーバー → Amazon EC2
		ファイル保存 → Amazon S3	
		
		などを数分で作成できます。
		料金は 使った分だけ支払う（Pay-as-you-go） モデルです。
	
	
	
	
	
# 	On-premiseとは
	
	**オンプレミス（On-premise）**とは、
		企業が自社内でITインフラを管理する方法です。
	
	
	## 会社は以下を自分で行います：
	
		サーバー購入
		データセンター設置
		ハードウェア保守
		ネットワーク管理

		つまり、
			すべて自分で管理するインフラ環境です。
	
	
	
	
# 	Cloud vs On-premise 比較
	
| 項目     | Cloud  | On-premise |
| ------ | ------ | ---------- |
| 初期コスト  | 低い     | 高い         |
| 拡張性    | 簡単     | 難しい        |
| サーバー作成 | 数分     | 数週間        |
| ハード管理  | クラウド会社 | 自社         |
| スケール   | 自動可能   | 手動         |

	
	
	
	
# 	Cloudを使う理由
	企業がクラウドを使う主な理由：
	
	
	1️ コスト削減（Cost saving）
	2️ スケーラビリティ（Scalability）
	3️ スピード（Speed）
	4️ グローバルインフラ（Global infrastructure）
