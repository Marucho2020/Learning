# Bản chất: Tại sao EBS nhanh hơn EFS? 


	Câu trả lời ngắn gọn:
		EBS gần như nói chuyện trực tiếp với ổ đĩa, còn EFS phải đi qua thêm một tầng File Server trên mạng.

	Đó là toàn bộ bản chất.


# Kiến trúc thật bên dưới

	## EBS
EC2
↓
NVMe Driver
↓
Nitro Card
↓
Amazon EBS Storage Server


Đối với Linux:
	EBS gần như xuất hiện thành:
	/dev/nvme0n1
	Kernel coi nó là:
	Block Device thật.
	Database cực thích điều này.





	## EFS
		Application
		↓
		POSIX
		↓
		NFS Client
		↓
		TCP/IP
		↓
		Network
		↓
		NFS Server
		↓
		Metadata Server
		↓
		Distributed Storage

		NFS protocol
		Network
		Metadata
		Permission
		Lock
		File system coordination

		=> tất cả đều tốn thời gian.




# Block Storage vs File Storage
	## Block Storage (EBS)

Disk

+----+
|001 |
+----+

+----+
|002 |
+----+

+----+
|003 |
+----+

Database hỏi:
	Cho tôi block số 251
	Xong.
	Không quan tâm file.
	Không quan tâm folder.
	Không quan tâm tên.


	## File Storage (EFS)
		File System phải làm rất nhiều việc.
	
	
/var/lib/mysql/
↓
ibdata1
↓
inode
↓
permission
↓
metadata
↓
block location	
	
	
	Muốn đọc một page của database:

		Phải trải qua:
		File name
		↓
		Directory lookup
		↓
		inode lookup
		↓
		Permission
		↓
		Metadata
		↓
		Block mapping
		↓
		Read
		Nhiều bước hơn.
			
			
	
# 	Database thực sự cần gì?

	Ví dụ PostgreSQL.	
	
	Muốn đọc:

		page 9812
			Nó không cần:
				picture.jpg
			Không cần:
				folder
			Không cần:
				permission
			Nó chỉ cần:
				Block 9812
				
		Đây là lý do toàn bộ database trên thế giới đều thích Block Device.
			
	

# IOPS là gì?

	IOPS
	=
	Input Output Operations Per Second

Ví dụ:



	## EBS io2

		Có thể đạt hàng chục nghìn đến hàng trăm nghìn IOPS (tùy cấu hình).
		Trong khi EFS được tối ưu cho:
			Many clients
			↓
			Many files
			↓
			Shared storage

		Không phải:
			Ultra Fast Random IO



# Một ví dụ dễ hiểu 

	EBS

		Giống như:
			Bạn có két tiền trong phòng.

		Muốn lấy:
			Mở két
			↓
			Lấy tiền

		Nhanh.
		
		
		
		
	EFS

		Giống như:
			Tiền ở ngân hàng.
			Bạn phải:		
		
			Đi ngân hàng
			↓
			Xếp hàng
			↓
			Kiểm tra CMND
			↓
			Kiểm tra tài khoản
			↓
			Lấy tiền		
				
		An toàn.
		Nhiều người dùng được.
		Nhưng chậm hơn.
