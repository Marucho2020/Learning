# 1. Cost Explorer là gì?
	AWS Cost Explorer là dịch vụ dùng để phân tích, trực quan hóa và dự đoán chi phí AWS.

	Nói đơn giản:

	Billing cho bạn biết đã tốn bao nhiêu tiền.

	Còn Cost Explorer trả lời:
		Tiền đang đi đâu?
		Dịch vụ nào tốn nhất?
		EC2 nào đắt nhất?
		Region nào tiêu nhiều nhất?
		Team nào đang dùng nhiều nhất?
		Tháng sau sẽ khoảng bao nhiêu tiền?

	Cost Explorer không phải công cụ tối ưu chi phí, mà là công cụ phân tích chi phí.
	Đây là điểm AWS rất thích hỏi.

# 4. Forecast (Dự đoán chi phí)
	Đây là tính năng rất hay xuất hiện trong đề thi.



		Ví dụ

		Ngày
			15/30

		Đã dùng
			$500

		Cost Explorer sẽ dự đoán
		Cuối tháng
			≈ $1000

		AWS dùng Machine Learning trên lịch sử usage.
		Không phải dự đoán chính xác tuyệt đối.




# 5. Group By

	Đây là phần quan trọng.
	Có thể Group theo

| Group By       | Ý nghĩa                         |
| -------------- | ------------------------------- |
| Service        | EC2, S3, Lambda...              |
| Region         | Tokyo, Ohio...                  |
| Linked Account | Trong AWS Organizations         |
| Tag            | Project, Team, CostCenter       |
| Usage Type     | Data Transfer, Compute Hours... |
| Instance Type  | t3.micro, m7i.large...          |



	Ví dụ

	Group By
	↓
	Tag
	↓
	Project
	sẽ ra
	Project A
	$300
	Project B
	$900

# 8. Mapping với các dịch vụ AWS

| Dịch vụ                   | Vai trò                             |
| ------------------------- | ----------------------------------- |
| Cost Explorer             | Phân tích chi phí                   |
| AWS Budgets               | Cảnh báo khi vượt ngân sách         |
| Cost & Usage Report (CUR) | Xuất dữ liệu chi tiết (CSV/Parquet) |
| Billing Dashboard         | Xem tổng tiền                       |
| Trusted Advisor           | Đề xuất tiết kiệm chi phí           |
| Compute Optimizer         | Đề xuất resize EC2, EBS, Lambda     |
| AWS Organizations         | Xem chi phí nhiều account           |



# 9. So sánh Cost Explorer vs Budgets vs CUR

| Dịch vụ           | Mục đích                      |
| ----------------- | ----------------------------- |
| Billing           | Tổng tiền                     |
| Cost Explorer     | Phân tích                     |
| Budgets           | Cảnh báo                      |
| CUR               | Dữ liệu chi tiết              |
| Compute Optimizer | Khuyến nghị tối ưu tài nguyên |
| Trusted Advisor   | Khuyến nghị tối ưu tổng thể   |



		Billing
			↓
		"Tôi đã tiêu bao nhiêu?"

		Cost Explorer
			↓
		"Tôi tiêu vào đâu?"

		Budget
			↓
		"Tôi sắp vượt ngân sách."

		CUR
			↓
		"Tôi muốn toàn bộ dữ liệu thô."

		Compute Optimizer
			↓
		"Resize cái EC2 này."

		Trusted Advisor
			↓
		"Đây là các cơ hội tiết kiệm."

# 12. Các bẫy (Trap)

| Đề bài                         | Đáp án đúng       | Bẫy thường gặp     |
| ------------------------------ | ----------------- | ------------------ |
| Phân tích chi phí              | Cost Explorer     | Chọn Budgets       |
| Cảnh báo vượt ngân sách        | AWS Budgets       | Chọn Cost Explorer |
| Xuất dữ liệu chi tiết          | CUR               | Chọn Cost Explorer |
| Khuyến nghị resize EC2         | Compute Optimizer | Chọn Cost Explorer |
| Khuyến nghị tiết kiệm tổng thể | Trusted Advisor   | Chọn Cost Explorer |
| Xem tổng hóa đơn               | Billing Dashboard | Chọn Cost Explorer |

# 13. Trade-off

| Cost Explorer mạnh ở                             | Hạn chế                                         |
| ------------------------------------------------ | ----------------------------------------------- |
| Phân tích chi phí trực quan                      | Không tự tối ưu tài nguyên                      |
| Forecast chi phí                                 | Chỉ là dự đoán, không chính xác tuyệt đối       |
| Group theo nhiều chiều (Service, Tag, Region...) | Không thay đổi cấu hình AWS                     |
| So sánh xu hướng theo thời gian                  | Không gửi cảnh báo (việc này thuộc AWS Budgets) |
