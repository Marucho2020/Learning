#	Khái niệm 
	Components là các khối code độc lập, tái sử dụng, bao gồm template, logic, và style, tương tự một ứng dụng Vue nhỏ.
	Mục đích: Chia nhỏ giao diện thành các phần riêng biệt (như button, header, form), dễ quản lý và tái sử dụng.
	Cách định nghĩa: Dùng app.component (global) hoặc trong file .vue (Single-File Component - SFC).
