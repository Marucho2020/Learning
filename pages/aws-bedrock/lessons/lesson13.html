Observability & Evaluation
Theo dõi Agent bằng CloudWatch, Trace, đánh giá Accuracy, Latency, Cost và Debug workflow. 


Agentic System Design
Thiết kế kiến trúc doanh nghiệp sử dụng Agent: phân chia nhiệm vụ, Human-in-the-loop, Retry, Fallback, Event-driven AI. 



// Nâng cao 
ReAct (Reason + Act) – nền tảng tư duy của Agent.
Function Calling / Tool Use – cơ chế LLM quyết định gọi công cụ.
Model Context Protocol (MCP) – chuẩn kết nối Agent với công cụ và dịch vụ.
Human-in-the-Loop (HITL) – khi nào Agent phải chuyển cho con người.
Long-term Memory vs Short-term Memory – các loại bộ nhớ của Agent.
Reflection / Self-Correction – Agent tự đánh giá và sửa kết quả.
Workflow vs Autonomous Agent – khác biệt giữa workflow cố định và Agent tự quyết định.
Deterministic Logic vs AI Reasoning – khi nào dùng code, khi nào dùng LLM.
Cost Optimization – giảm token, giảm số lần gọi model, cache, batching.
Enterprise Agent Patterns – các mẫu kiến trúc như Supervisor–Worker, Planner–Executor, Router Agent, Reviewer Agent.
