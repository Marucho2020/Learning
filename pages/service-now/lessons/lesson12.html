Lesson 12 -- ⚔️ Normalize vs Standardize -//

			
			
# 🧱 1. Normalize (Chuẩn hóa dữ liệu theo cấu trúc)


	👉 Mục tiêu:
		Loại bỏ trùng lặp
		Tách data thành các bảng hợp lý
		Đảm bảo data consistency

	👉 Bản chất:
		Thiết kế database schema



	## Ví dụ (rất thực tế trong ServiceNow) 

		❌ Không normalize:
			Incident Table
			- caller_name
			- caller_department
			👉 lặp lại hàng nghìn lần

		✅ Normalize:

			User Table
			- name
			- department

			Incident Table
			- caller_id (reference → User)

		👉 dùng reference field + dot walk

	⚠️ Nếu không normalize:
		data inconsistency (department bị sai)
		update nightmare
		report sai




# 🎯 2. Standardize (Chuẩn hóa cách biểu diễn dữ liệu)


	👉 Mục tiêu:
		Data đồng nhất format
		Dễ xử lý / search / automation

	👉 Bản chất:
		Chuẩn hóa giá trị, không phải cấu trúc



	Ví dụ:
		❌ Không standardize:
			"VN"
			"Vietnam"
			"Việt Nam"

		✅ Standardize:
			"VN"

		Trong ServiceNow:
			dùng:
				Choice field
				Dictionary
				Data policy
				UI policy



# 🔥 So sánh thẳng

| Aspect        | Normalize         | Standardize          |
| ------------- | ----------------- | -------------------- |
| Mục tiêu      | cấu trúc DB       | format dữ liệu       |
| Tác động      | schema            | value                |
| Công cụ       | table, reference  | choice, policy       |
| Sai → hậu quả | DB rối, duplicate | data bẩn, khó search |
| Level         | kiến trúc         | vận hành             |



# 🧠 Góc nhìn senior (quan trọng) 

❗ 1. Normalize sai → hệ thống chết chậm
	duplicate data
	không scale được
	automation fail


❗ 2. Standardize sai → AI / report vô dụng
	cùng 1 thứ nhưng 10 cách viết
	query không match


# 🧨 3. Cái bẫy lớn 


	Nhiều dev:
		standardize thay vì normalize

	Ví dụ:
		tạo choice field cho department ❌
		thay vì reference User table ✅

	👉 đây là design ngu (thẳng luôn)



# 🚀 4. Trong ServiceNow — cách làm đúng


	Normalize:
	dùng:
		reference field
		CMDB
		relational structure



	Standardize:
			dùng:
				choice list
				data policy
				script enforce format




# 🧭 Kết luận
	Normalize = thiết kế xương sống hệ thống
	Standardize = giữ data sạch để dùng được 


		👉 Thiếu 1 trong 2:

		hoặc hệ thống không scale
		hoặc data không usable
