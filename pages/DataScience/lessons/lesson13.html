# 3. C√°ch T√≠nh Percentiles Th·ªß C√¥ng (Chi Ti·∫øt Nh∆∞ Chuy√™n Gia)
	ƒê·ªÉ t√≠nh percentile th·ª© P:

	T√≠nh v·ªã tr√≠: S·ª≠ d·ª•ng c√¥ng th·ª©c n·ªôi suy tuy·∫øn t√≠nh (linear interpolation, m·∫∑c ƒë·ªãnh trong NumPy):

	Index = (P / 100) * (n - 1), trong ƒë√≥ n l√† s·ªë l∆∞·ª£ng d·ªØ li·ªáu.
N·∫øu index l√† s·ªë nguy√™n k, gi√° tr·ªã = data[k].

	V√≠ d·ª• c·ª• th·ªÉ 1: T√≠nh 10% percentile c·ªßa Max_Pulse (nh∆∞ nhi·ªám v·ª• trong b√†i)

		n = 10, P = 10.

		k = 0, f = 0.9.
		Gi√° tr·ªã = 120 + 0.9 * (120 - 120) = 120 + 0 = 120.
		√ù nghƒ©a: 10% c√°c bu·ªïi t·∫≠p c√≥ Max_Pulse ‚â§ 120 bpm. (Kh·ªõp v·ªõi v√≠ d·ª• trong b√†i.)



	V√≠ d·ª• c·ª• th·ªÉ 2: T√≠nh 25% percentile c·ªßa Average_Pulse (ƒë·ªÉ minh h·ªça)

		D·ªØ li·ªáu Average_Pulse: 80, 85, 90, 95, 100, 105, 110, 115, 120, 125 (ƒë√£ s·∫Øp x·∫øp).
		n = 10, P = 25.
		Index = 0.25 * 9 = 2.25.
		k = 2, f = 0.25.
		Gi√° tr·ªã = 90 + 0.25 * (95 - 90) = 90 + 0.25 * 5 = 90 + 1.25 = 91.25.
		√ù nghƒ©a: 25% bu·ªïi t·∫≠p c√≥ Average_Pulse ‚â§ 91.25 bpm. (Trong full data c·ªßa W3Schools, gi√° tr·ªã n√†y l√† 100, c√≥ l·∫Ω do d·ªØ li·ªáu l·ªõn h∆°n v·ªõi nhi·ªÅu gi√° tr·ªã l·∫∑p ho·∫∑c ph√¢n b·ªë kh√°c.)
