# 1. Tại Sao Học Data Science Và Phân Tích Dữ Liệu?

	Tầm quan trọng: Như bài viết nói, "Data rules the world" (Dữ liệu thống trị thế giới). Trong thời đại số, các công ty như Google, Amazon hay Netflix sử dụng dữ liệu để dự đoán hành vi khách hàng, tối ưu hóa sản phẩm, và tăng doanh thu. # 2. Học Qua Ví Dụ: "Try it Yourself" 

	Bài viết nhấn mạnh học qua thực hành với editor trực tuyến. Bạn có thể chỉnh sửa mã Python và xem kết quả ngay. Đây là cách học hiệu quả như chuyên gia: không chỉ đọc lý thuyết mà phải "làm" để hiểu.

	## Ví Dụ Chính Trong Bài: Hồi Quy Tuyến Tính Với Dữ Liệu Sức Khỏe

		Mã ví dụ sử dụng dữ liệu từ file "data.csv" (giả sử chứa dữ liệu về nhịp tim trung bình và lượng calo đốt cháy). Mục tiêu: Phân tích mối quan hệ giữa nhịp tim (Average_Pulse) và calo đốt cháy (Calorie_Burnage) qua hồi quy tuyến tính, rồi vẽ biểu đồ.
	Mã đầy đủ từ bài:

import pandas as pd
import matplotlib.pyplot as plt
from scipy import stats

full_health_data = pd.read_csv("data.csv", header=0, sep=",")

x = full_health_data["Average_Pulse"]
y = full_health_data["Calorie_Burnage"]

slope, intercept, r, p, std_err = stats.linregress(x, y)

def myfunc(x):
    return slope * x + intercept

mymodel = list(map(myfunc, x))

plt.scatter(x, y)
plt.plot(x, mymodel)
plt.ylim(ymin=0, ymax=2000)
plt.xlim(xmin=0, xmax=200)
plt.xlabel("Average_Pulse")
plt.ylabel("Calorie_Burnage")
plt.show()


# Giải Thích Chi Tiết Từng Dòng (Như Chuyên Gia Phân Tích):

	## Import thư viện: 
	
		- import pandas as pd: Pandas là thư viện xử lý dữ liệu dạng bảng (như Excel). Dùng để đọc file CSV và thao tác dữ liệu.
		- import matplotlib.pyplot as plt: Matplotlib để vẽ biểu đồ, giúp visualize dữ liệu.
		- from scipy import stats: SciPy cung cấp công cụ thống kê, ở đây dùng cho hồi quy tuyến tính (linear regression).

	## Đọc dữ liệu:
		full_health_data = pd.read_csv("data.csv", header=0, sep=","): Đọc file CSV vào DataFrame (bảng dữ liệu). header=0 nghĩa là dòng đầu tiên là tiêu đề cột; sep="," là dấu phân cách là dấu phẩy.

		Minh họa: Giả sử "data.csv" có nội dung như sau (dữ liệu mẫu tôi tạo để minh họa, vì bài không cung cấp):

Average_Pulse,Calorie_Burnage
80,300
85,330
90,350
95,380
100,400
105,430
110,450

	## Trích xuất biến
		x = full_health_data["Average_Pulse"]: Lấy cột nhịp tim làm biến độc lập (independent variable).
		y = full_health_data["Calorie_Burnage"]: Lấy cột calo làm biến phụ thuộc (dependent variable).
		Khái niệm: Trong phân tích dữ liệu, chúng ta thường tìm mối quan hệ giữa các biến. Ở đây, giả sử nhịp tim cao hơn dẫn đến đốt calo nhiều hơn.


	## Thực hiện hồi quy tuyến tính:
		slope, intercept, r, p, std_err = stats.linregress(x, y): Sử dụng hàm linregress từ SciPy để tính:

			- slope: Độ dốc của đường thẳng (hệ số hồi quy, cho biết y thay đổi bao nhiêu khi x tăng 1 đơn vị).
			- intercept: Giá trị y khi x=0 (điểm chặn).
			- r: Hệ số tương quan (correlation coefficient, từ -1 đến 1; gần 1 nghĩa là tương quan mạnh dương).
			- p: Giá trị p-value (kiểm định thống kê; nhỏ hơn 0.05 nghĩa là mối quan hệ có ý nghĩa).
			- std_err: Sai số chuẩn của slope.

		Lý thuyết chi tiết: Hồi quy tuyến tính là mô hình toán học dạng y = slope * x + intercept + error. Nó dự đoán y dựa trên x bằng cách tìm đường thẳng phù hợp nhất với dữ liệu (sử dụng phương pháp bình phương nhỏ nhất - least squares)

	Ví dụ minh họa: Với dữ liệu mẫu trên, chạy mã sẽ cho slope ≈ 6 (mỗi nhịp tim tăng thêm đốt 6 calo), intercept ≈ -180, r ≈ 0.99 (tương quan rất mạnh).


	## Xây dựng hàm dự đoán
		
		def myfunc(x): return slope * x + intercept: Hàm tính y dự đoán từ x.
		
		mymodel = list(map(myfunc, x)): Áp dụng hàm cho mọi x để tạo danh sách y dự đoán.


	## Vẽ biểu đồ:

		- plt.scatter(x, y): Vẽ scatter plot (điểm dữ liệu thực tế).
		- plt.plot(x, mymodel): Vẽ đường hồi quy (dự đoán).
		- plt.ylim(ymin=0, ymax=2000) và plt.xlim(xmin=0, xmax=200): Đặt giới hạn trục y và x để biểu đồ rõ ràng.
		- plt.xlabel("Average_Pulse") và plt.ylabel("Calorie_Burnage"): Đặt nhãn trục.
		- plt.show(): Hiển thị biểu đồ.

		Minh họa hình ảnh: Biểu đồ sẽ cho thấy các điểm dữ liệu nằm gần đường thẳng, chứng tỏ mô hình tốt. Nếu r cao, đường thẳng dốc lên, nghĩa là nhịp tim cao hơn → calo đốt nhiều hơn.
