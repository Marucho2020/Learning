# 1. Tạo DataFrame Với Pandas (Create a DataFrame with Pandas) 
	
	Định nghĩa chính: DataFrame là cách biểu diễn dữ liệu có cấu trúc (structured representation of data), tương tự bảng cơ sở dữ liệu (database table) trong bài trước. Bài đưa ra ví dụ tạo DataFrame với 3 cột và 5 hàng từ số liệu giả lập.


	## Mã ví dụ từ bài 
	
import pandas as pd
d = {'col1': [1, 2, 3, 4, 7], 'col2': [4, 5, 6, 9, 5], 'col3': [7, 8, 12, 1, 11]}
df = pd.DataFrame(data=d)
print(df)		


	## Giải thích chi tiết và chuyên sâu
		Import Pandas: import pandas as pd – Pandas là thư viện chính (đã giới thiệu ở bài trước) để xử lý dữ liệu dạng bảng. pd là alias thông dụng để gọi nhanh hàm của Pandas.

		Tạo dictionary: d = {'col1': [1, 2, 3, 4, 7], 'col2': [4, 5, 6, 9, 5], 'col3': [7, 8, 12, 1, 11]} – Dictionary d chứa dữ liệu với key là tên cột (col1, col2, col3) và value là list các giá trị (mỗi list là một cột).

		Tạo DataFrame: df = pd.DataFrame(data=d) – Hàm pd.DataFrame() chuyển dictionary thành DataFrame, với các cột được đặt tên theo keys và giá trị là các rows. Chữ DataFrame cần viết đúng (chữ D và F in hoa).
	
		In DataFrame: print(df) – Hiển thị bảng dữ liệu.

   col1  col2  col3
0     1     4     7
1     2     5     8
2     3     6    12
3     4     9     1
4     7     5    11


		Phân tích: DataFrame có 3 cột (col1, col2, col3) và 5 hàng (rows 0-4). Các số 0-4 là index (chỉ số hàng), bắt đầu từ 0 theo chuẩn Python. Đây là structured data, tương tự bảng sức khỏe trong bài trước.
	
	
		Kết nối với bài trước: Trong bài về database table, dữ liệu sức khỏe (Duration, Average_Pulse, Calorie_Burnage, v.v.) được lưu trong DataFrame bằng pd.read_csv(). Ở đây, bài tạo DataFrame từ dictionary, nhưng ý tưởng giống nhau: tổ chức dữ liệu thành bảng.


import pandas as pd
data = {
    'Duration': [30, 30, 45, 45, 45, 60, 60, 60, 75, 75],
    'Average_Pulse': [80, 85, 90, 95, 100, 105, 110, 115, 120, 125],
    'Calorie_Burnage': [240, 250, 260, 270, 280, 290, 300, 310, 320, 330]
}
df_health = pd.DataFrame(data=data)
print(df_health)

9        75            125              330

	Phân tích: Đây là cách tạo DataFrame tương tự file data.csv trong bài đầu tiên, sẵn sàng cho phân tích (như hồi quy tuyến tính).

	Ứng dụng thực tế: Trong phân tích doanh thu, Data Scientist tạo DataFrame từ dữ liệu bán hàng (cột: Product, Price, Quantity) để tính tổng doanh thu hoặc tìm patterns mua sắm.


	Lời khuyên chuyên gia: Dictionary là cách linh hoạt để tạo DataFrame nhỏ. Với dữ liệu lớn, dùng pd.read_csv() hoặc pd.read_sql() (từ database). Luôn kiểm tra DataFrame bằng df.head() để xem 5 hàng đầu tiên.


# 2. Diễn Giải Output (Interpreting the Output)
	
	Định nghĩa chính: Output của DataFrame hiển thị cột (column names: col1, col2, col3) và chỉ số hàng (row indices: 0-4). Trong Python, chỉ số bắt đầu từ 0.

	## Giải thích chi tiết và chuyên sâu

		Cột: Tên cột (col1, col2, col3) là labels, tương tự header trong database table (bài trước: Duration, Average_Pulse, v.v.). Chúng đại diện cho variables (biến số) – mỗi cột là một thuộc tính có thể đo lường.
		
		Hàng: Chỉ số 0-4 là row indices, giúp truy cập dữ liệu. Index 0 tương ứng với hàng đầu tiên ([1, 4, 7]). Python dùng 0-based indexing, khác với một số ngôn ngữ như R (1-based).
		
		## Minh họa chuyên sâu: Với DataFrame sức khỏe
		
		df_health['Average_Pulse'] truy cập cột Average_Pulse (output: Series [80, 85, ...]).
		
		df_health.iloc[0] truy cập hàng 0 (output: Duration=30, Average_Pulse=80, Calorie_Burnage=240).
		
		Ý nghĩa: Index giúp lọc dữ liệu nhanh, ví dụ: df_health[df_health['Duration'] > 60] trả về các hàng có Duration > 60 (rows 8, 9).
		
		
	## Ví dụ minh họa cụ thể: Tính trung bình Calorie_Burnage cho các buổi tập dài hơn 45 phút: 
		
		mean_calories = df_health[df_health['Duration'] > 45]['Calorie_Burnage'].mean()
	print(f"Mean Calorie Burnage for Duration > 45: {mean_calories}")	
		
		
		Ứng dụng thực tế: Index rất quan trọng trong time-series data (dữ liệu theo thời gian), như phân tích giá cổ phiếu hàng ngày, với index là ngày.
		
		Lưu ý chuyên sâu: Index có thể tùy chỉnh (ví dụ: đặt cột ngày làm index bằng df.set_index('Date')). Nếu sai index, phân tích sẽ lỗi – luôn kiểm tra bằng df.index.
		
		
		
		
# 3. Đếm Số Cột và Hàng (Counting Columns and Rows)

	Định nghĩa chính: Sử dụng df.shape[1] để đếm số cột và df.shape[0] để đếm số hàng. Bài nhấn mạnh không nên đếm thủ công với dataset lớn để tránh sai sót

	Mã ví dụ từ bài:
		count_column = df.shape[1]
		print(count_column)  # Output: 3

		count_row = df.shape[0]
		print(count_row)  # Output: 5
	
		Thuộc tính shape: df.shape trả về tuple (số hàng, số cột). Ví dụ: (5, 3) nghĩa là 5 hàng, 3 cột.

		Tại sao không đếm thủ công?

Với dataset nhỏ (như ví dụ: 5 hàng, 3 cột), đếm tay khả thi. Nhưng với big data (hàng triệu hàng), đếm thủ công dễ sai và mất thời gian.
shape là hàm built-in, đảm bảo chính xác, đặc biệt khi làm việc với dữ liệu động (dynamic data) từ API hoặc database.


	## Ví dụ minh họa cụ thể: Với DataFrame sức khỏe: 
	
print(df_health.shape)  # Output: (10, 3)
print(f"Number of rows: {df_health.shape[0]}")  # 10
print(f"Number of columns: {df_health.shape[1]}")  # 3
