Lesson 186 -- Pipeline -//

# 1. Pipeline trong lập trình nói chung

	Khái niệm: Pipeline = nhiều bước xử lý nối tiếp nhau, đầu ra của bước này → đầu vào của bước kế.

	Giống như dây chuyền sản xuất: nguyên liệu đi qua nhiều công đoạn, cuối cùng ra sản phẩm.

	Ưu điểm:
		Dễ tách nhỏ logic.
		
		Dễ tái sử dụng từng bước.
		
		Dễ thay đổi/thêm bước mới.


# 2. Pipeline trong Java (liên hệ Functional Interface) 
	Functional Interfaces + chaining = cách Java hiện thực hóa pipeline.

Function trim = String::trim;
Function toUpper = String::toUpperCase;

Function pipeline = trim.andThen(toUpper);
System.out.println(pipeline.apply("   hello ")); // "HELLO"

➡️ Input " hello " đi qua bước trim → kết quả "hello" → tiếp tục qua toUpper → "HELLO".



	## Predicate pipeline

Predicate hasEmail = e -> e.getEmail() != null;
