Lesson 7 -- Tham chiếu Garbage Collection -//

public class Main {
    public static void main(String[] args) {
        Person p1 = new Person(); // Tạo đối tượng mới
        Person p2 = p1;           // p1 và p2 đều tham chiếu đến đối tượng Person

        p1 = null;  // p1 không còn tham chiếu đến đối tượng nữa
        // Tuy nhiên, p2 vẫn tham chiếu đến đối tượng Person
        System.out.println(p2.age); // p2 vẫn có thể truy cập đối tượng
    } 
}

Đối tượng sẽ bị garbage collection dọn dẹp khi không còn biến nào tham chiếu đến nó.
Trong trường hợp bạn gán p1 = null;, đối tượng Person không bị dọn dẹp ngay lập tức vì p2 vẫn tham chiếu đến nó. Chỉ khi p2 cũng được gán null hoặc không còn tham chiếu đến đối tượng nữa, thì đối tượng này mới không có tham chiếu nào và có thể bị Garbage Collector dọn dẹp.

Tóm lại: Chỉ khi không còn "sợi dây" nào kết nối đến đối tượng nữa (tức là tất cả các tham chiếu đều trỏ về null hoặc không còn tham chiếu đến đối tượng đó), thì Garbage Collector mới có thể dọn dẹp bộ nhớ cho đối tượng đó.
