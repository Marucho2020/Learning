Lesson 180 -- General-purpose Functional Interface 汎用関数インターフェース -//

	
# Khái niệm 
	汎用 (はんよう / han’yō) = “phổ dụng, dùng chung, general-purpose”
	関数インターフェース = “functional interface” (tức interface chỉ có đúng 1 abstract method – dùng cho lambda).

→ Ghép lại: 汎用関数インターフェース = General-purpose Functional Interfaces in Java.


	Java 8 giới thiệu package java.util.function với cả một họ functional interfaces “phổ dụng” mà ai cũng có thể tái sử dụng, thay vì tự định nghĩa interface riêng. Đây chính là 汎用関数インターフェース.



# Một số nhóm chính: 

	## Function 
	
		- Đầu vào: T
		- Đầu ra: R
		- Dùng để chuyển đổi dữ liệu.


	## Consumer
	
		- Đầu vào: T
		- Không trả về gì (void).
		- Dùng để thực thi hành động có side-effect.	
	
	
	
	## Supplier 
	
		Không đầu vào.
		Trả về T.
		Dùng để sinh ra dữ liệu.	
			
	
	## Predicate
		Đầu vào: T
		Trả về boolean.
		Dùng để test điều kiện.
	
	
	## UnaryOperator và BinaryOperator
		- Phiên bản đặc biệt của Function nhưng input/output cùng kiểu.
		- Dùng nhiều trong xử lý toán học, reduce, accumulate.
	
#3. Tại sao gọi là “汎用 (phổ dụng, dùng chung)”?

	Trước Java 8, dev phải tự định nghĩa các interface như MyFilter, MyTransformer chỉ để truyền lambda.
	
	Sau Java 8, chỉ cần dùng mấy functional interfaces chuẩn → code thống nhất, giảm boilerplate.
	
	Các lib/framework (Spring, Reactor, Stream API…) đều dựa trên mấy interface này.


# 4. Ý nghĩa ở level senior

	Đây không chỉ là “lambda sugar” mà là cách Java chuẩn hóa mô hình lập trình hàm (functional programming model).

	Khi hiểu đúng, bạn sẽ biết:

		- Khi nào nên chọn Function vs Predicate.
		- Cách compose chúng: andThen, compose, negate, and, or...
		- Cách tận dụng với Stream API để biến code từ imperative → declarative
	
	Ở tầm kiến trúc: 汎用関数インターフェース cho phép bạn viết API mở rộng dễ hơn. Ví dụ:	

	public  List mapList(List input, Function mapper) {
    return input.stream().map(mapper).toList();
}



汎用関数インターフェース = General-purpose Functional Interface trong Java.

Nó là nhóm interface chuẩn (Function, Predicate, Supplier, Consumer, …) trong java.util.function.

Giá trị: chuẩn hóa functional programming trong Java, giúp code ngắn gọn, tái sử dụng, dễ kết hợp.
